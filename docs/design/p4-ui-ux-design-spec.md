# P4 页面视觉与交互设计规范 - 自愿打赏版

> 目标：强引导自愿打赏、零门槛启用VIP、阳光少年风格  
> 适用范围：`/`、`/login`、`/register`、`/sponsor`（打赏页）及 `/chat` 入口引导  
> 设计基调：**"Sunshine Boy / White Samoyed"** —— 阳光、干净、温暖、真诚  
> 核心色彩：天空蓝 + 麦穗金 + 暖白  
> **重要原则**：打赏完全自愿，不打赏也能启用VIP，但页面要极度吸引用户打赏

## 页面 4：打赏页 `/sponsor` ⭐核心页面

### 设计目标

**极度吸引用户打赏，同时明确"自愿、不打赏也能用"**

- 页面要有强烈的情感吸引力（真诚、温暖）
- 收款码要醒目、易操作
- "我已打赏"按钮要突出，让用户感觉"完成了一件好事"
- 底部小字明确"自愿、不影响使用"建立信任

### 整体氛围

**"阳光下的真诚请求"** —— 温暖、真诚、不强迫但极度吸引

- 背景：暖白色 → 淡淡的金色光晕（营造温暖感）
- 主视觉：大号emoji或插图 + 真诚文案
- 收款码：清晰展示，方便扫码
- CTA：只有一个主按钮，清晰明确

   ↓
4. 用户点击"我已打赏，启用VIP"
   - 按钮Loading态（1秒）
   - 调用现有VIP启用逻辑
   - 显示成功Toast："VIP已启用，感谢支持！"
   ↓
5. 跳转 return_to（默认/chat）
```

**按钮状态**
```
默认：麦穗金渐变，发光阴影

Loading：Spinner动画，文案"启用中..."

## 设计原则总结

### ✅ 要做的

1. **极度吸引**：打赏页要有强烈的情感号召，但不能强迫
2. **真诚透明**：明确说明"不打赏也能用"，建立信任
3. **温暖阳光**：麦穗金+天空蓝+暖白，邻家少年感
4. **单一CTA**：打赏页只有一个主按钮，清晰明确
5. **移动端优先**：大按钮、大二维码、易操作

### ❌ 严禁的

1. **强迫感**：倒计时、禁用按钮、遮罩层
2. **误导文案**："未付款无法使用"、"支付验证中"
3. **蓝紫色调**：与"Sunshine Boy"气质冲突
4. **暗黑风格**：不符合白厄阳光少年形象
5. **复杂流程**：多步骤、表单、审核
